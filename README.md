# 广西南宁市智慧城市数据可视化大屏 (187项目)

这是一个基于 **HTML/Vanilla JS**、**CSS3** 和 **ECharts** 开发的现代化数据可视化大屏系统。项目专注于展示广西南宁市各行政区的核心运行指标，具备极致的视觉美感与交互性能。

## 🌟 项目亮点

- **双击即读**：经过重构，项目不再依赖本地服务器环境。您可以**直接双击 `index.html`** 在浏览器中查看完整效果。
- **极致美学设计**：深蓝科技感配色的暗黑模式，采用毛玻璃效果（Glassmorphism）与多重阴影，营造高级感。
- **动态数字孪生**：中心大屏展示南宁各行政区的地理节点，配合动态飞线与涟漪效果，模拟城市智慧网络。
- **全量数据覆盖**：包含南宁 10 个以上核心区县的详细人口、经济、交通等多维度模拟数据。
- **零配置环境**：通过 CDN 引入 ECharts 核心库与词云插件，无需本地安装依赖即可运行。

## 🛠️ 技术栈

- **图表库**: [ECharts v5.5.1](https://echarts.apache.org/) (CDN 加载)
- **核心逻辑**: Vanilla JavaScript (原生脚本，兼容非模块环境)
- **样式**: 原生 CSS3 (Flexbox & Grid)

## 🚀 启动方式

### 方式一：直接运行 (推荐)
直接进入项目目录，找到 `index.html` 并**双击打开**。

### 方式二：工程化开发
虽然支持直接打开，但项目依然保留了 `package.json`。如果您需要进行深度开发：
1. `npm install`
2. `npm run dev`

## 📊 展示维度

- **中间内容**：南宁市智慧运行核心节点图，展示行政区间的逻辑关联与动态飞线。
- **左侧面板**：区域人口密度分布（渐变柱状图）、年度经济增长趋势（面积折线图）、交通出行占比（精细环形图）。
- **右侧面板**：资源调度评估（雷达图）、城市实时热词（动态词云）、系统负荷状态（三段色仪表盘）。

## 📝 技术细节

1. **Tooltip 修复**：针对 `axis` 和 `item` 两种触发模式进行了兼容性重写，彻底解决了切换图表时可能出现的数据显示 `undefined` 问题。
2. **连接路径算法**：中心地图通过计算地理坐标间的拓扑关系，动态生成从核心节点向四周辐射的 `lines` 轨迹。
3. **自适应处理**：全局监听 `resize` 事件，确保大屏在不同分辨率、缩放比例下均能保持像素级对齐。
